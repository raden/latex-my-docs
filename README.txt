CARA MEMBINA PDF
Fail .tex yang dikompil memerlukan anda mempunyai pakej texlive pada sistem anda. Untuk memudahkan urusan kompilasi, sila install texlive-latex-extra

Dalam kebanyakan hal saya menggunakan Ubuntu, jadi dengan,

sudo apt-get install texlive-latex-extra membolehkan apt-get menyelesaikan masalah dependency yang berkaitan

PAKEJ ARABTEX dan ALQALAM
Pakej arabtex datang dengan texlive-lang-arab
Pakej alqalam datang dengan alqalam

sudo apt-get install  texlive-lang-arab  alqalam 

untuk menyelesaikan masalah kebergantungan ini
